# Proyecto MMORPG Medieval - Three.js

## Visión General
Un MMORPG de fantasía medieval basado en navegador utilizando Three.js. El juego cuenta con 3 facciones en guerra, zonas dedicadas y un sistema de conquista PvP.

## Facciones
1. **Orden del Sol (Facción 1)**
   - Estilo: Paladines, Clérigos, Luz.
   - Zona Inicial: Valles Soleados.
   - Capital: Bastión del Amanecer.
2. **Pacto de la Sombra (Facción 2)**
   - Estilo: Pícaros, Nigromantes, Oscuridad.
   - Zona Inicial: Bosque de las Brumas.
   - Capital: Torre del Crepúsculo.
3. **Alianza de la Naturaleza (Facción 3)**
   - Estilo: Druidas, Cazadores, Elementos.
   - Zona Inicial: Arboleda Ancestral.
   - Capital: Corazón Verde.

## Zonas
- **Zonas Seguras**: Áreas iniciales y de leveo para cada facción (PvE).
- **Zona de Guerra (Centro)**: Área PvP abierta con fuertes conquistables que otorgan beneficios a la facción controladora.

## Clases y Progresión
- Sistema de niveles (1-60).
- Árboles de habilidades ramificados.
- Arquetipos clásicos: Tanque, DPS (Cuerpo a cuerpo/Rango), Sanador.

## Tech Stack
- **Frontend**: Three.js, React (para UI), Vite.
- **Backend**: Node.js, Socket.io / Colyseus.
- **Base de Datos**: MongoDB / PostgreSQL.
